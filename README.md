# ERP Employee Management System (MySQL)

A comprehensive MySQL database  to manage employee and department information. This demonstrates database design, data manipulation, and advanced SQL concepts commonly used in real-world applications.

## Features

* Database and table creation
* CRUD Operations (Create, Read, Update, Delete)
* Data filtering using WHERE and LIKE clauses
* Aggregate Functions (COUNT, SUM, AVG, MAX, MIN)
* GROUP BY and HAVING clauses
* INNER JOIN operations
* Subqueries and Nested Queries
* UNION and Set Operations
* Views
* Indexing
* Stored Procedures
* Triggers
* Common Table Expressions (CTEs)
* Window Functions (ROW_NUMBER, RANK, DENSE_RANK)
* Transactions (COMMIT and ROLLBACK)

## Technologies Used

* MySQL
* SQL

## Learning Outcomes

This helped in understanding relational database design, query optimization, data retrieval techniques, and advanced SQL concepts used in enterprise applications.
